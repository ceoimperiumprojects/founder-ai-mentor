What is it

AI Mentor Powered by Real Startup Knowledge

Not another chatbot. This is a RAG-powered system that lets Claude search through 10 curated startup books to give you grounded, actionable advice.

📚

10 Curated Books

From Hormozi's $100M Offers to Thiel's Zero to One — the most impactful startup books, semantically indexed.

🧠

Semantic Search

Uses sentence-transformers and ChromaDB for deep semantic chunking — finds the right answer, not just keyword matches.

⚡

MCP Integration

Runs as a Model Context Protocol server. Claude Desktop connects directly and queries your knowledge base in real-time.

How it works

Up and Running in 3 Steps

Clone the repo, add your books, and start asking Claude for startup advice.

1

Clone the Repo

Get the project and install dependencies.

git clone && pip install -r requirements.txt
2

Add Your Books

Drop PDFs or EPUBs into data/raw/ and run the ingestion script to build the knowledge base.

python scripts/ingest_pdf.py --dir data/raw/
3

Ask Claude

Connect Claude Desktop via MCP and start asking real startup questions.

"How do I validate my startup idea?"
